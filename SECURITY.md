# Security Policy

## Supported Versions

Only the latest version of ZethraOS is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.0  | :white_check_mark: |

## Reporting a Vulnerability

We take the security of ZethraOS seriously. If you believe you have found a security vulnerability, please report it to us by following these steps:

1. **Do not disclose publicly**: Please do not open public issues for security vulnerabilities.
2. **Email us**: Send a detailed report to security@zethraos.com (or use GitHub Private Vulnerability Reporting).
3. **Wait for response**: We will acknowledge your report within 48 hours and provide a timeline for a fix.

For more information on our commitment to security, visit [zethraos.com/security](https://zethraos.com/security).
